# Catalog

Catalog 是数据存储与管理的核心组件，专为多模态数据管理设计。在数据驱动的业务场景中，Catalog 可整合多种数据源，提升数据管理效率，同时确保数据的安全性与可用性。

## 原始数据卷

原始数据卷用于存放用户上传的未处理非结构化数据，作为数据处理的基础存储单元。

处理数据卷用于存放经过数据清洗、转换等处理后的非结构化数据，便于后续分析与应用。
